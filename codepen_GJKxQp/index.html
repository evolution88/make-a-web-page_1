Important Notes

Internet and Basics of HTML

Definitions

World Wide Web - collection of HTML documents
HTML - HyperText Markup Language
Hyper Links - links between webpages
Browser - a program that runs on your computer to display files found on the web Internet - the world’s largest computer network
Servers - computers that host the files that make up the web
HTTP - the main protocol of the web

HTML Components

HTML is made up of the following components:
- text content - what you see
- markup - what it looks like
- references to other documents - e.g. images & videos
- links to other pages

Basic HTML Vocabulary

Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags (called "void" tags) do not require a closing tag (like the br tag).
Element: An HTML element refers to everything within a set of opening and closing tags.
Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to the Udacity URL.

Inline vs. Block Elements

HTML elements are either inline or block.
Block elements form an "invisible box" around the content inside of them.

Purposes of HTML, Javascript and CSS

HTML - structure (walls / living room)
CSS - style (color / decorations)
Javascript - interactive components (TV remote)

Structured Documents with HTML

Tree-like Structure

A tree structure or tree diagram is a way of representing the hierarchical nature of a structure in a graphical form. It is named a "tree structure" because the classic representation resembles a tree, even though the chart is generally upside down compared to an actual tree, with the "root" at the top and the "leaves" at the bottom.

The Box Model

According to the box model concept, every element on a page is a rectangular box and may have width, height, padding, borders, and margin.
The core of the box is defined by the width and height of an element, which may be determined by the display property, by the contents of the element, or by specified width and height properties. padding and then border expand the dimensions of the box outward from the element’s width and height. Lastly, any marginwe have specified will follow the border.

Text Editors

Text editor options:

CSS Styles and HTLM Structure

How does CSS control HTML

Styling can be added to HTML elements in 3 ways:
Inline - using a style attribute in HTML elements
Internal - using a style element in the HTML head section
External - using one or more external CSS files
The most common way to add styling, is to keep the styles in separate CSS files

Avoiding Errors

When programmers don't avoid repetition, they will often have to do the same thing over and over. This may mean copy and pasting certain style attributes into many HTML elements or rewriting the same code (sometimes with minor differences) many times. This can lead to errors when the programmer decides to make a change to something. If they don't dilligently make that same change everywhere the repeated code appears, problems will arise.
